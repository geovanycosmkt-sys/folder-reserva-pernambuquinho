                                                        ## Formato do Folder

                                                        ### Tipo
                                                        - Folder sanfona (accordion/Z-fold)
                                                        - - 6 paineis, 2 dobras
                                                          - - Impresso frente e verso (3 paineis cada lado)
                                                           
                                                            - ### Formato Fisico (Die-Cut)
                                                            - - Formato MAMADEIRA — TODOS os 6 paineis devem ter o formato de mamadeira
                                                              - - Descricao: corpo retangular com cantos muito arredondados + projecao arredondada no topo (o "bico")
                                                                - - Parece uma mamadeira de bebe: corpo largo embaixo, bump arredondado em cima
                                                                  - - Ver arquivo referencia-formato.jpg no repositorio
                                                                   
                                                                    - ### Dimensoes
                                                                    - - Altura: 297mm (A4)
                                                                      - - Largura por painel: ~70mm (210mm / 3)
                                                                        - - Largura total aberto: 210mm
                                                                          - - Resolucao: 300dpi
                                                                            - - Cores de impressao: CMYK
                                                                              - - Sangria: 3mm
